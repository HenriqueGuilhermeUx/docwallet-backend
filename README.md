# DocWallet Backend

Backend standalone do DocWallet para Render.

Inclui API de login, documentos, contratos, certificados e validação blockchain.

Arquivos principais:

- `app.py`: API Flask
- `requirements.txt`: dependências Python
- `render.yaml`: blueprint para Render
- `.env.example`: modelo de variáveis de ambiente
- `start.sh`: script de inicialização

Para testar após o deploy, acesse `/api/health`.
